# SemCap-CPP

This repository contains code accompanying the paper **Semantic Map Sharing
and Capability-Aware Coverage Planning for AI-Native 6G Robotic
Coordination**.

## Available code

The [`dataset_processing`](dataset_processing/) directory contains the label
mappings and preprocessing scripts used to construct the common semantic
dataset. Its README provides links to the three source datasets, the expected
directory structure, and instructions for running the processing pipeline.

## Planned release

The capability-aware coverage-planning implementation and the remaining
evaluation scripts will be uploaded upon acceptance of the paper.
